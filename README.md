## Installation

### From The Unity Asset Store (Recomended) (Not Avalible Yet)
* To Start Go To The [Unity Asset Store](https://assetstore.unity.com/packages/slug/362682) And Click `Add To my Assets`
* Then In Unity Open the `Package Manager` Window
* Look For `Bolts Tools`
* Click On The `Download` Button
* Then Click On The `Import` Button
 
### From Github
* To Start Go To [Releases](https://github.com/Bolt-Bug/Boolts-Tools/releases)
* Download The Latest
* In Unity, Drag The `unitypackage` File Into Your Project
* Click The `Import` Button

## Prerelase
To Download Prerelases:
* Download The Project And Open It In Unity
* Right Click In The Project Window And Click On `Export Package`
* Then Select The `Editor` Folder, `Bolts Tools` Folder And The `Resources` Folder
